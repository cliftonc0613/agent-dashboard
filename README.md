# agent-dashboard

Rebel Alliance command center — the Next.js 15 + Convex dashboard for the autonomous site-building pipeline.

## Stack

- Next.js 15.5.15 (pinned)
- React 19.2.5 (pinned)
- Tailwind CSS v4
- Convex (wired in Plan 01-02)
- shadcn/ui (wired in Plan 01-03)

## Local Development

```bash
npm run dev          # Next.js dev server on :3000
npx convex dev       # Convex dev backend (run in separate terminal)
```
